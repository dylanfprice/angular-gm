angulargmContainer
service in module angulargm

Description

A container which maps mapIds to google.maps.Map instances, and additionally allows getting a promise of a map for custom configuration of the map.

If you want a handle to the map, you should use the getMapPromise(mapId) method so you can guarantee the map will be initialized. For example,

angular.module('myModule').

run(function(angulargmContainer) {
  var gmapPromise = angulargmContainer.getMapPromise('myMapid');

  gmapPromise.then(function(gmap) {
    // google map configuration here
  });
});
